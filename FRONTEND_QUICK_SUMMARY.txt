## 🚀 Ejecutar

```bash
# Terminal 1
cd backend/mod_busqueda && python app.py

# Terminal 2
cd backend && python app.py

# Terminal 3
cd frontend && npm install && npm start
```

**Abre**: http://localhost:3000

**⚠️ Importante**: Agregar `CORS(app)` en `backend/app.py`
